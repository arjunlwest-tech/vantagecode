# Vantage Coding ⚡

A gamified coding quest platform by **Vantage Suites**. Level up your programming skills through interactive quests, language tracks, projects, and achievements.

## Features

- 🎮 **Gamified XP System** — Earn XP, level up, and track your progress
- ⚔️ **12 Daily Quests** — Complete coding challenges across categories
- 💻 **30+ Programming Languages** — From JavaScript to Assembly
- 📚 **Learning Tracks** — Beginner to Advanced paths for each language
- 🚀 **Project Blueprints** — Step-by-step project guides with task breakdowns
- 🧠 **Interactive Coach** — Describe an idea and get an instant development plan
- 🎨 **Live Code Previews** — See rendered code samples per language
- 🏆 **12 Achievements** — Unlock badges as you progress
- 📊 **Leaderboard** — Compete against other coders
- 📝 **Learning Notes** — Jot down and save what you've learned
- 🔥 **Streak Tracking** — Keep your daily streak alive
- ✨ **Particle Effects** — Premium animated background
- 💾 **Persistent Progress** — All data saved to localStorage

## Tech Stack

- Pure HTML, CSS, JavaScript (zero dependencies)
- Google Fonts (Inter + JetBrains Mono)
- Google AdSense integration
- Canvas-based particle system

## Deploy to Vercel

1. Push this repo to GitHub
2. Go to [vercel.com](https://vercel.com)
3. Click "Import Project" → Select your GitHub repo
4. Click "Deploy" — that's it!

Or use the Vercel CLI:

```bash
npx vercel
```

## Deploy via GitHub

1. Create a new repository on GitHub
2. Run:

```bash
git init
git add .
git commit -m "Initial commit"
git branch -M main
git remote add origin https://github.com/YOUR_USERNAME/vantagecoding.git
git push -u origin main
```

3. Connect to Vercel for automatic deployments on every push.

## Local Development

Simply open `index.html` in your browser. No build step required.

```bash
# Or use a local server
npx serve .
```
